# 🐍 Snake Game in C++

A console-based implementation of the classic Snake Game built using C++. This project focuses on core programming concepts such as arrays, loops, functions, conditional statements, and real-time user input handling.

## 🚀 Features

- Interactive snake movement using keyboard controls
- Real-time food generation
- Score tracking system
- Collision detection with walls and snake body
- Game-over condition handling
- Simple and lightweight console interface

## 🎮 Controls

| Key | Action |
|---|---|
| W | Move Up |
| S | Move Down |
| A | Move Left |
| D | Move Right |
| X | Exit Game |
